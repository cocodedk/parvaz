چکیده

شرح مأموریت

پرواز یک اپلیکیشن VPN اندروید برای کاربران ایرانی بدون دانش فنی است. یک فرد آشنا به فن — خویشاوند، داوطلب، فعال — یک Cloudflare Worker یک‌فایلی را روی حساب کاربری خودش مستقر می‌کند؛ کاربر یک لینک parvaz:// را از تلگرام دریافت می‌کند، QR را اسکن می‌کند یا روی «چسباندن» می‌زند، و تمام ترافیک گوشی از طریق Worker آن شخص عبور می‌کند. TLS SNI به یک دامنهٔ محبوب کلودفلر، Host به *.workers.dev شما.

یک APK. یک لمس. به صورت پیش‌فرض فارسی. بدون Play Store، بدون ثبت‌نام، بدون تله‌متری. رابط کاربری Kotlin + Compose، هستهٔ Go برای SOCKS5، سرور ~۵۰ خط JavaScript. همه متن‌باز با مجوز MIT.

مشخصات بدنهٔ پرواز

ویژگی‌ها
۰۱دامین فرانتینگTLS SNI یک دامنهٔ پرترافیک کلودفلر است. HTTP Host آدرس *.workers.dev شماست. DPI فقط «ترافیک کلودفلر» را می‌بیند؛ کلودفلر بر اساس Host واقعی به صورت داخلی مسیریابی می‌کند.
۰۲یک لمس، فارسی اوللینک parvaz:// را از تلگرام بچسبانید یا QR را اسکن کنید → Connect را بزنید. VpnService همهٔ ترافیک گوشی را می‌گیرد. رابط به صورت پیش‌فرض فارسی است؛ انگلیسی فقط بازنویسی.
۰۳APK متن‌بازرابط Kotlin + Compose، هستهٔ تعبیه‌شدهٔ Go برای SOCKS5، Worker ~۵۰ خطی JavaScript. بدون وابستگی انحصاری. خارج از Play Store — فقط F-Droid و sideload.
۰۴دستنامهٔ میدانیزیبایی‌شناسی NOTAM. تایپوگرافی Vazirmatn (فارسی)، Redaction (نمایشی)، JetBrains Mono (کد). رابط فارسی از پایه — RTL درجه یک، نه یک الحاقیه.
۰۵عبور خام TCPWorker از API cloudflare:sockets connect() استفاده می‌کند — سوکت TCP خروجی واقعی باز می‌کند. بایت‌ها بدون دست‌کاری عبور می‌کنند — بدون MITM، بدون نصب CA، همهٔ اپ‌ها کار می‌کنند.
۰۶بدون تله‌متریبدون آنالیتیکس، بدون پینگ به‌روزرسانی، بدون گزارش کرش. کلید دسترسی شما در EncryptedSharedPreferences می‌ماند. هیچ تماس شبکه‌ای به خارج، جز خود Worker.

پشتیبانی زمینی

موردنیاز · upstream

پرواز از بک‌اند Apps Script در MasterHttpRelayVPN فاصله گرفت — UrlFetchApp.fetch() مبتنی بر URL است و نمی‌تواند بایت‌های HTTPS را بدون MITM عبور دهد، و اندروید ۷+ به CAهای نصب‌شدهٔ کاربر به صورت پیش‌فرض اعتماد نمی‌کند. نیمهٔ سرور جدید یک Cloudflare Worker ~۵۰ خطی است که از API cloudflare:sockets برای TCP خروجی خام استفاده می‌کند. برای استقرار ۵ مرحله‌ای، worker/README.md را ببینید.

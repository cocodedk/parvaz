§۱.۰

چکیده

شرح مأموریت

پرواز یک تونل مرورگر اندروید با اولویت فارسی برای کاربران ایرانی بدون دانش فنی است. یک فرد آشنا به فن — خویشاوند، داوطلب، فعال — پروندهٔ upstream Code.gs متعلق به MasterHttpRelayVPN را روی حساب Google خود مستقر می‌کند. کاربر لینک parvaz:// را از تلگرام دریافت می‌کند، QR را اسکن یا «چسباندن» را می‌زند، یک بار گواهی MITM را از تنظیمات اندروید نصب می‌کند، و Connect را لمس می‌کند. ترافیک Chrome و Firefox از طریق Apps Script عبور می‌کند — TLS SNI www.google.com، HTTP Host script.google.com.

محدودهٔ پرواز فقط مرورگر است. Instagram / Telegram / WhatsApp / اپ‌های بانکی به گواهی نصب‌شدهٔ کاربر اعتماد نمی‌کنند و تونل نمی‌شوند — این محدودیت عمومی MITM در اندروید است، نه باگ پرواز. معماری با پورت MasterHttpRelayVPN-RUST هم‌تراز است؛ تفاوت پرواز در زیبایی‌شناسی NOTAM، رابط پیش‌فرض فارسی، و راه‌اندازی ساده‌تر است.

اعلان به خلبانان

فضای هوایی محدود
هشدار · فضای هوایی محدود NOTAM A0001/26
شرایط استفادهٔ Google Apps Script این کاربرد را ممنوع اعلام کرده است. Code.gs را فقط روی حساب Google خودتان مستقر کنید — مستقرسازی متمرکز یا توزیع یک deployment مشترک ممنوع است. فقط برای استفادهٔ شخصی، تحقیقاتی و آموزشی. اگر با پیامدهای حقوقی آشنا نیستید، آن را مستقر نکنید.
§۴.۰

مشخصات بدنهٔ پرواز

ویژگی‌ها
۰۱دامین فرانتینگTLS SNI می‌گوید www.google.com. HTTP Host می‌گوید script.google.com. DPI فقط فرانت را می‌بیند؛ Google بر اساس Host واقعی به صورت داخلی مسیریابی می‌کند.
۰۲راه‌اندازی فارسی اوللینک parvaz:// را از تلگرام بچسبانید یا QR را اسکن کنید → یک بار گواهی MITM را از تنظیمات نصب کنید → Connect بزنید. رابط به صورت پیش‌فرض فارسی است؛ انگلیسی فقط بازنویسی.
۰۳فقط مرورگر — صادقانهChrome و Firefox به‌درستی تونل می‌شوند. Instagram، Telegram، WhatsApp، اپ‌های بانکی به صورت پیش‌فرض به گواهی نصب‌شدهٔ کاربر اعتماد نمی‌کنند و تونل نمی‌شوند. محدودیت عمومی MITM اندروید — برای آن‌ها xray + VPS جداگانه نیاز است.
۰۴مسیر سریع SNI-rewriteدامنه‌های متعلق به Google (google.com، youtube.com، fonts.googleapis.com و …) مستقیماً با بازنویسی SNI تونل می‌شوند و از Apps Script عبور نمی‌کنند. بدون MITM، بدون سهمیهٔ Apps Script، سرعت کامل.
۰۵APK متن‌بازرابط Kotlin + Compose، هستهٔ تعبیه‌شدهٔ Go برای SOCKS5. بدون وابستگی انحصاری. خارج از Play Store — فقط F-Droid و sideload.
۰۶بدون تله‌متریبدون آنالیتیکس، بدون پینگ به‌روزرسانی، بدون گزارش کرش. کلید دسترسی شما در EncryptedSharedPreferences می‌ماند. هیچ تماس شبکه‌ای به خارج، جز خود رله.

§۵.۵

نصب گواهی · در تنظیمات

دستی · یک بار · بخش پیچیده

اندروید ۱۱ به بعد اجازه نمی‌دهد یک برنامه گواهی CA را خودش نصب کند. وقتی در پرواز روی «بازکردن تنظیمات» می‌زنید، برنامه فایل parvaz-ca.crt را در پوشهٔ Downloads قرار می‌دهد و سپس تنظیمات اندروید را تا حد امکان نزدیک به نقطهٔ نصب باز می‌کند. از آن‌جا به بعد را — فقط یک بار — به‌صورت دستی پیش می‌برید. پیش‌نیاز: گوشی باید قفل صفحه (PIN، الگو یا رمز) داشته باشد. اندروید بدون قفل صفحه، نصب گواهی را نمی‌پذیرد.

# سامسونگ (Galaxy S23 / A52s / مشابه — One UI 6.x): Settings → Security and privacy (امنیت و حریم خصوصی) → More security settings (تنظیمات امنیتی بیشتر) → Install from device storage (نصب از حافظهٔ دستگاه) → CA certificate (گواهی CA) → فایل parvaz-ca.crt را از Downloads انتخاب کنید → روی هشدار، Install anyway را بزنید (به‌هرحال نصب کن)
proc · cert.001 · pixel / aosp Android 14 / 15
# Pixel / اندروید استاندارد (نسخه‌های ۱۴ و ۱۵ — برچسب‌ها یکسان): Settings → Security & privacy (امنیت و حریم خصوصی) → More security & privacy (امنیت و حریم خصوصی بیشتر) → Encryption & credentials (رمزگذاری و گواهی‌نامه‌ها) → Install a certificate (نصب یک گواهی) → CA certificate (گواهی CA) → فایل parvaz-ca.crt را از Downloads انتخاب کنید → روی هشدار، Install anyway را بزنید (به‌هرحال نصب کن)

مسیر را پیدا نمی‌کنید؟ در تنظیمات، روی آیکن ذره‌بین جست‌وجو بزنید و عبارت CA certificate را تایپ کنید. اولین نتیجه، مسیر منوی دقیق روی پوستهٔ گوشی شما را نشان می‌دهد.

دربارهٔ پنجرهٔ هشدار. اندروید پنجره‌ای با عنوان «Your data won't be private» («اطلاعات شما خصوصی نخواهد ماند») نمایش می‌دهد، با متنی طولانی و دو دکمه: Don't install و Install anyway. این هشدار اصولاً درست است: یک CA سوءاستفاده‌گر می‌تواند ترافیک شما را شنود کند. در مورد پرواز، CA همان لحظهٔ اول روی گوشی شما ساخته می‌شود و کلید خصوصی آن هرگز از filesDir خارج نمی‌شود. Install anyway را بزنید.

بعدش چه می‌شود. به پرواز برگردید (با اشارهٔ بازگشت یا تعویض اپ). کنترل‌گر onResume برنامه، فروشگاه CAهای کاربری را پیمایش می‌کند، با SHA-256 تطبیق می‌دهد، و در کمتر از یک ثانیه به مرحلهٔ اجازهٔ VPN می‌رود. اگر تأیید موفق نشد، صفحه دکمهٔ «تلاش دوباره» نمایش می‌دهد — معمولاً به دلیل تأخیر در به‌روزرسانی فروشگاه است و یک بار تلاش کافی است.

اپ‌هایی که هنوز کار نمی‌کنند. مرورگرها (Chrome، Firefox) به CAهای کاربر اعتماد می‌کنند و تونل می‌شوند. Instagram، Telegram، WhatsApp، اپ‌های بانکی به‌صورت پیش‌فرض گواهی نصب‌شدهٔ کاربر را رد می‌کنند و تونل نمی‌شوند — این محدودیت عمومی اندروید برای هر پراکسی MITM است. اگر به آن اپ‌ها نیاز دارید، پرواز را با xray + VPS ترکیب کنید.

پشتیبانی زمینی

موردنیاز · upstream

پرواز فقط نیمهٔ کلاینت MasterHttpRelayVPN را جایگزین می‌کند. سرور Apps Script همان Code.gs upstream است — بدون تغییر روی حساب Google خودتان مستقر کنید. اگر به ظرافت‌های UX پرواز (فارسی پیش‌فرض، زیبایی‌شناسی NOTAM) نیاز ندارید، پورت رست MasterHttpRelayVPN-RUST امروز با APKهای آماده و راهنمای کامل انگلیسی + فارسی در دسترس است.

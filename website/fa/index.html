§۱.۰

چکیده

شرح مأموریت

پرواز یک تونل مرورگر اندروید با اولویت فارسی برای کاربران ایرانی بدون دانش فنی است. یک فرد آشنا به فن — خویشاوند، داوطلب، فعال — پروندهٔ upstream Code.gs متعلق به MasterHttpRelayVPN را روی حساب Google خود مستقر می‌کند. کاربر لینک parvaz:// را از تلگرام دریافت می‌کند، QR را اسکن یا «چسباندن» را می‌زند، یک بار گواهی MITM را از تنظیمات اندروید نصب می‌کند، Firefox Nightly را نصب می‌کند و یک تنظیم در about:config را روشن می‌کند، و Connect را لمس می‌کند. ترافیک Firefox Nightly از طریق Apps Script عبور می‌کند — TLS SNI www.google.com، HTTP Host script.google.com.

محدودهٔ پرواز فقط Firefox Nightly است. Chrome استاندارد و Firefox استاندارد در اندروید ۷+ گواهی نصب‌شدهٔ کاربر را نادیده می‌گیرند و HTTPS را تونل نمی‌کنند. Instagram / Telegram / WhatsApp / اپ‌های بانکی نیز گواهی کاربر را به همین دلیل رد می‌کنند. این‌ها محدودیت‌های عمومی MITM در اندروید است، نه باگ پرواز. معماری با پورت MasterHttpRelayVPN-RUST هم‌تراز است؛ تفاوت پرواز در زیبایی‌شناسی NOTAM، رابط پیش‌فرض فارسی، و راه‌اندازی ساده‌تر است.

اعلان به خلبانان

فضای هوایی محدود
هشدار · فضای هوایی محدود NOTAM A0001/26
شرایط استفادهٔ Google Apps Script این کاربرد را ممنوع اعلام کرده است. Code.gs را فقط روی حساب Google خودتان مستقر کنید — مستقرسازی متمرکز یا توزیع یک deployment مشترک ممنوع است. فقط برای استفادهٔ شخصی، تحقیقاتی و آموزشی. اگر با پیامدهای حقوقی آشنا نیستید، آن را مستقر نکنید.
§۴.۰

مشخصات بدنهٔ پرواز

ویژگی‌ها
۰۱دامین فرانتینگTLS SNI می‌گوید www.google.com. HTTP Host می‌گوید script.google.com. DPI فقط فرانت را می‌بیند؛ Google بر اساس Host واقعی به صورت داخلی مسیریابی می‌کند.
۰۲راه‌اندازی فارسی اوللینک parvaz:// را از تلگرام بچسبانید یا QR را اسکن کنید → یک بار گواهی MITM را از تنظیمات نصب کنید → Connect بزنید. رابط به صورت پیش‌فرض فارسی است؛ انگلیسی فقط بازنویسی.
۰۳فقط Firefox Nightly — صادقانهFirefox Nightly با تنظیم security.enterprise_roots.enabled = true به‌درستی HTTPS را تونل می‌کند. Chrome استاندارد، Firefox استاندارد، Brave، Edge — همگی در اندروید ۷+ گواهی کاربر را نادیده می‌گیرند. Instagram، Telegram، WhatsApp، اپ‌های بانکی نیز همین‌طور. محدودیت عمومی MITM اندروید — برای آن‌ها xray + VPS نیاز است.
۰۴مسیر سریع SNI-rewriteدامنه‌های متعلق به Google (google.com، youtube.com، fonts.googleapis.com و …) مستقیماً با بازنویسی SNI تونل می‌شوند و از Apps Script عبور نمی‌کنند. بدون MITM، بدون سهمیهٔ Apps Script، سرعت کامل.
۰۵APK متن‌بازرابط Kotlin + Compose، هستهٔ تعبیه‌شدهٔ Go برای SOCKS5. بدون وابستگی انحصاری. خارج از Play Store — فقط F-Droid و sideload.
۰۶بدون تله‌متریبدون آنالیتیکس، بدون پینگ به‌روزرسانی، بدون گزارش کرش. کلید دسترسی شما در EncryptedSharedPreferences می‌ماند. هیچ تماس شبکه‌ای به خارج، جز خود رله.

نصب گواهی · در تنظیمات

گام ۱ از ۲ · یک بار · ~۳۰ ثانیه

در پرواز، «بازکردن تنظیمات» را بزنید. پرواز فایل parvaz-ca.crt را در پوشهٔ Downloads قرار می‌دهد و تنظیمات اندروید را باز می‌کند. سپس مسیر مربوط به گوشی خود را دنبال کنید:

proc · cert.001 · samsung One UI 6 / Android 14
# سامسونگ (Galaxy): Settings → Security and privacy (امنیت و حریم خصوصی) → More security settings (تنظیمات امنیتی بیشتر) → Install from device storage (نصب از حافظهٔ دستگاه) → CA certificate (گواهی CA) → فایل parvaz-ca.crt را از Downloads انتخاب کنید → PIN یا اثر انگشت خود را وارد کنید → روی هشدار، Install anyway را بزنید (به‌هرحال نصب کن)
proc · cert.001 · pixel / aosp Android 14 / 15
# Pixel / اندروید استاندارد: Settings → Security & privacy (امنیت و حریم خصوصی) → More security & privacy (امنیت و حریم خصوصی بیشتر) → Encryption & credentials (رمزگذاری و گواهی‌نامه‌ها) → Install a certificate (نصب یک گواهی) → CA certificate (گواهی CA) → فایل parvaz-ca.crt را از Downloads انتخاب کنید → PIN یا اثر انگشت خود را وارد کنید → روی هشدار، Install anyway را بزنید (به‌هرحال نصب کن)

پیش‌نیاز: گوشی شما باید قفل صفحه (PIN، الگو یا رمز) داشته باشد. اندروید بدون قفل صفحه، نصب گواهی را نمی‌پذیرد.

مسیر را پیدا نمی‌کنید؟ در تنظیمات، عبارت CA certificate را جست‌وجو کنید. اولین نتیجه مسیر دقیق روی گوشی شما را نشان می‌دهد.

هشدار را با خیال راحت رد کنید. اندروید پنجره‌ای با عنوان «Your data won't be private» («اطلاعات شما خصوصی نخواهد ماند») نمایش می‌دهد. این هشدار به‌طور کلی درست است — یک CA سوءاستفاده‌گر می‌تواند ترافیک شما را شنود کند. در مورد پرواز، CA همان لحظهٔ اول روی گوشی شما ساخته می‌شود و کلید خصوصی آن هرگز از حافظهٔ خصوصی برنامه خارج نمی‌شود. Install anyway را بزنید.

سپس به پرواز برگردید. اشارهٔ بازگشت بزنید. پرواز در حدود یک ثانیه گواهی را تشخیص می‌دهد و به صفحهٔ بعد می‌رود. اگر نرفت، یک بار «تلاش دوباره» را بزنید.

§۵.۶

نصب Firefox Nightly · فعال کردن یک تنظیم

گام ۲ از ۲ · یک بار · ~۱ دقیقه

Chrome استاندارد و Firefox استاندارد گواهی‌ای را که الآن نصب کردید نادیده می‌گیرند — اندروید ۷+ مرورگرهای غیرسیستمی را از اعتماد به CA کاربر منع می‌کند و هیچ تنظیمی برای تغییر این رفتار وجود ندارد. تنها مرورگر بدون نیاز به root که CA کاربر را می‌پذیرد Firefox Nightly است، آن هم فقط پس از فعال کردن یک تنظیم.

proc · browser.001 · firefox-nightly یک بار · ~۱ دقیقه
# ۱. Firefox Nightly را نصب کنید F-Droid → جست‌وجو: Firefox Nightly Play Store → جست‌وجو: Firefox Nightly for Developers # ۲. Firefox Nightly را باز کنید. در نوار آدرس، این را بچسبانید: about:config → روی هشدار «به‌هرحال ادامه بده» را بزنید # ۳. در کادر جست‌وجوی بالای صفحه، این را تایپ کنید: security.enterprise_roots.enabled → روی ردیف بزنید تا مقدارش به true تغییر کند # ۴. Firefox Nightly را کاملاً ببندید (از Recents حذف کنید) و دوباره باز کنید.

تمام شد. در Firefox Nightly به‌صورت معمول مرور کنید — صفحات HTTPS از طریق پرواز عبور می‌کنند. سایر مرورگرها همچنان دامنه‌های متعلق به Google (YouTube، Google Search، Google Maps) را از مسیر سریع SNI-rewrite بدون نیاز به گواهی بارگذاری می‌کنند. باقی همه چیز نیاز به Firefox Nightly دارد.

چرا فقط Nightly؟ Firefox فروشگاه گواهی ریشهٔ مخصوص خود را داخل APK جای می‌دهد؛ تنها کانال Nightly تنظیم security.enterprise_roots.enabled را در دسترس قرار می‌دهد، که به Firefox می‌گوید گواهی‌های فروشگاه CA کاربر اندروید را نیز اعتماد کند. Firefox استاندارد، Beta و Focus این تنظیم را ندارند.

اپ‌هایی که هنوز کار نمی‌کنند. Instagram، Telegram، WhatsApp، اپ‌های بانکی، استریمینگ — همه به‌صورت پیش‌فرض گواهی نصب‌شدهٔ کاربر را رد می‌کنند. محدودیت عمومی اندروید است. اگر به آن‌ها نیاز دارید، پرواز را با xray + VPS ترکیب کنید.

پشتیبانی زمینی

موردنیاز · upstream

پرواز فقط نیمهٔ کلاینت MasterHttpRelayVPN را جایگزین می‌کند. سرور Apps Script همان Code.gs upstream است — بدون تغییر روی حساب Google خودتان مستقر کنید. اگر به ظرافت‌های UX پرواز (فارسی پیش‌فرض، زیبایی‌شناسی NOTAM) نیاز ندارید، پورت رست MasterHttpRelayVPN-RUST امروز با APKهای آماده و راهنمای کامل انگلیسی + فارسی در دسترس است.

§1.0

Abstract

Mission brief

Parvaz is an Android VPN app built for Iranian users with no technical background. A helper deploys a one-file Cloudflare Worker; the user receives a parvaz:// link over Telegram, scans a QR or taps Paste, and every byte of phone traffic tunnels through their own Worker — TLS SNI to a popular Cloudflare-hosted site, HTTP Host to the user's *.workers.dev deployment.

One APK. One tap. Farsi-first. No Play Store, no account, no telemetry. Kotlin + Compose front end. Go SOCKS5 sidecar. ~50-line JavaScript Worker. All open source, MIT-licensed.

Notice to airmen

Restricted airspace
Caution · Restricted airspace NOTAM A0001/26
Cloudflare's acceptable-use policy is strict about proxy services. Deploy the Worker to your own Cloudflare account only — do not centralize deployments, do not commercialise, do not share your worker with strangers you can't vouch for. Personal, research, and educational use only.
§4.0

Airframe specifications

Features
01Domain frontingTLS SNI says a popular Cloudflare-hosted site. HTTP Host says your *.workers.dev. DPI sees "Cloudflare traffic"; Cloudflare routes by the real Host internally.
02One-tap, Farsi-firstPaste a parvaz:// URL or scan a QR from Telegram → tap Connect. VpnService captures all phone traffic. UI defaults to Persian; English is the override.
03Open source APKKotlin + Compose front end, embedded Go SOCKS5 sidecar, ~50-line Cloudflare Worker. No proprietary deps. Ships outside the Play Store — F-Droid and sideload only.
04Light-theme field manualNOTAM aesthetic. Vazirmatn (Persian), Redaction (display), JetBrains Mono (code). Persian UI built in — first-class RTL, not an afterthought.
05Raw TCP passthroughThe Worker uses cloudflare:sockets connect() to open a real outbound TCP socket. Bytes pass through opaque — no MITM, no CA install, every app works.
06Zero telemetryNo analytics, no auto-update pings, no crash reporting. Your access key stays in EncryptedSharedPreferences. No third-party network calls other than the Worker itself.

§7.0

Ground support

Required · upstream

Parvaz pivoted away from the MasterHttpRelayVPN Apps Script backend — UrlFetchApp.fetch() is URL-based HTTP and cannot tunnel opaque HTTPS without MITM, and Android 7+ won't trust user-installed CAs per-app. The new server half is a ~50-line Cloudflare Worker that uses the cloudflare:sockets API for raw outbound TCP. See worker/README.md for the 5-step deployment.
